Easily add customizable call to action blocks in your posts and pages with the Call To Action Customizable Block for Gutenberg.

== Description ==
The Call To Action Customizable Block for Gutenberg allows you to add eye-catching call to action (CTA) boxes in your posts and pages. With a clean design and a variety of customization options, this block helps you drive user engagement and conversions effectively.

= Features =
* **Customizable Title**: Add and style the title to grab attention.
* **Customizable Description**: Add and style the description to convey your message.
* **Customizable Button**: Add a button with customizable text and link to direct users to the desired action.
* **Color Options**: Customize the colors of the title, description, button text, and button background to match your site's theme.
* **Responsive Design**: Ensure your call to action blocks look great on all devices.
* **User-Friendly**: Integrated seamlessly into the Gutenberg editor for an intuitive user experience.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/call-to-action-customizable-block` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to the Gutenberg editor, and add the "Call To Action Customizable Block" to your post or page.

== Frequently Asked Questions ==

= How do I customize the colors of the block? =
In the block settings panel, you'll find options to change the colors of the title, description, and button. Simply choose your desired colors from the color pickers provided.

= Can I add multiple Call To Action blocks to a single post or page? =
Yes, you can add as many Call To Action blocks as you need to any post or page.

= Is this plugin compatible with all themes? =
The Call To Action Customizable Block is designed to work with all WordPress themes. However, some themes with unique CSS styles might require minor adjustments.

= What if I encounter an issue with the plugin? =
If you encounter any issues, please contact support through the WordPress plugin support forum. We'll be happy to assist you.
